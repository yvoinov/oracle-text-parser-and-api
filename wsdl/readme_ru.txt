      Автоматический запуск инстанса OC4J
      -----------------------------------

При   автоматическом   запуске   инстанса   OC4J  необходимо
установить  переменные  DISPLAY  (указывающую  на  сервер!),
JAVA_HOME  (/usr  на Солярис) и ORACLE_HOME, который смотрит
на OC4J_EXTENDED.

Имя владельца схемы в config.xml надо установить правильно,
как и пароль для коннекта. Кроме того, должна быть указана
корректная ссылка на инстанс базы данных.

  Для включения Oc4jMount для OHS (Apache) необходимо:
  ----------------------------------------------------

1. Изменить default-web-site.xml инстанса OC4J, указав явно
  (или изменив умолчание http) протокол ajp13 для порта 8888.
  (каталог config/oc4j).
2. Изменить  для  Апача  файл  mod_oc4j.conf   как  показано
   в файле mod_oc4j.conf данного архива (каталог config/ohs).
3. Активизировать автоматический запуск OC4J
   и перезапустить Апач (OHS) и OC4J.
